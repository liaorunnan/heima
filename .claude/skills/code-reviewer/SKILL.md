# 代码审查 Skill

## 何时使用

当需要执行以下任务时使用此 Skill：
- 审查代码质量
- 检测潜在 bug
- 识别安全漏洞
- 检查代码风格
- 验证最佳实践

## 审查步骤

### 1. 阅读代码
- 理解代码结构和逻辑
- 识别主要功能和组件

### 2. 安全检查
- 搜索硬编码密钥
- 检查 SQL 注入风险
- 验证输入验证

### 3. 错误处理
检查错误处理模式：
- 是否正确处理异常
- 是否有适当的验证
- 是否有用户输入过滤

### 4. 输出报告
整理发现的问题并提供修复建议

## 最佳实践

- 使用具体示例说明问题
- 提供可执行的修复建议
- 按严重程度分类问题
- 引用相关文档和标准
