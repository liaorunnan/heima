# 智慧教务管理系统 - 用户操作手册

## 1. 准备与启动
*   **界面描述**：浏览器将显示一个深色背景的登录窗口，屏幕中央是一个白色的卡片区域。

---

## 2. 系统登录
### 界面外观
*   屏幕中心为一个白色悬浮框，顶部有蓝色学士帽图标 。
*   包含两个输入框：“账号”、“密码”。
*   下方有一个**蓝色**的长条按钮，字样为 **“登录”**。

### 操作流程
1.  **输入账号**：在账号框输入 `admin`。
2.  **输入密码**：在密码框输入 `123456`。
3.  **点击登录**：鼠标左键点击**蓝色“登录”按钮**。
    *   **成功反馈**：登录遮罩层会逐渐变透明并消失，进入系统主界面。
    *   **失败反馈**：如果密码错误，浏览器顶部会弹出提示框“账号或密码错误”。

---

## 3. 主界面概览
### 界面外观
系统分为左右两栏布局：
*   **左侧菜单栏（深灰色）**：包含系统名称 “EduSystem” 和四个导航菜单（仪表盘、学生管理、课程安排、考勤记录）。当前选中的菜单背景会变深且左侧有一条**蓝色竖线**高亮。
*   **顶部栏（白色）**：左侧显示当前页面标题（如“仪表盘”），右侧显示当前登录用户（管理员）及圆形头像。
*   **右侧内容区（浅灰色）**：显示具体的操作内容。

### 退出登录
*   **位置**：左侧菜单栏最底部。
*   **操作**：点击**红色**的 **“退出登录”** 按钮。
*   **反馈**：页面刷新，重新回到登录界面。

---

## 4. 仪表盘模块
### 界面外观
*   顶部横向排列三个统计卡片：
    1.  **蓝色卡片**：显示“总学生数”及具体数字。
    2.  **绿色卡片**：显示“今日出勤率”。
    3.  **紫色卡片**：显示“今日课程数”。
*   下方为白色欢迎区域。

### 操作
*   该页面为**只读**展示页面，用于快速查看教务概况。系统启动后默认显示此页面。

---

## 5. 学生人员管理模块
### 界面外观
*   顶部左侧：一个白色的**搜索输入框**，提示文字“搜索姓名...”。
*   顶部右侧：一个**蓝色**按钮，字样为 **“+ 新增学生”**。
*   主体区域：学生名单按照 **年级（灰色横条）** 分组，年级下方按照 **班级（白色方块卡片）** 再次分组。

### 功能 A：新增学生
1.  **点击按钮**：点击右上角的**蓝色 “+ 新增学生”** 按钮。
2.  **弹出窗口**：屏幕中央弹出“新增学生”模态框。
3.  **填写信息**：
    *   **姓名**：输入学生名字（如：张无忌）。
    *   **年级**：下拉选择（高一/高二/高三）。
    *   **班级**：下拉选择（1班/2班/3班）。
4.  **保存**：点击模态框右下角的**蓝色 “保存”** 按钮。
5.  **反馈**：模态框关闭，页面右下角弹出“操作成功”提示，新学生卡片立即出现在列表中。

### 功能 B：搜索学生
1.  **输入关键字**：在左上角输入框输入学生姓名（如“张”）。
2.  **反馈**：输入的同时，下方列表会**实时过滤**，只显示名字中包含“张”的学生。清空输入框后恢复显示所有学生。

### 功能 C：编辑与删除
*   **编辑**：
    1.  在学生卡片右侧找到**蓝色小铅笔图标**并点击。
    2.  弹出窗口显示该学生当前信息。修改后点击**蓝色 “保存”** 按钮即可更新。
*   **删除**：
    1.  在学生卡片右侧找到**红色垃圾桶图标**并点击。
    2.  浏览器弹出确认框：“确定删除该学生吗...”。
    3.  点击“确定”后，该学生从列表中消失，右下角提示“删除成功”。

## 6. 课程安排模块
### 界面外观
*   顶部区域：
    *   左侧：文字“选择学生查看课表”，旁边是一个**下拉选择框**。
    *   右侧：一个**蓝紫色（Indigo色）**按钮，字样为 **“+ 添加/编辑课程”**。
*   主体区域：一个 8行（节次） x 5列（星期） 的网格表格。
    *   有课的格子显示为**浅紫色背景、深紫色文字**的胶囊状标签。
    *   无课的格子显示为灰色的短横线。

### 功能 A：查看课表
1.  **选择学生**：点击左上角的下拉框，选择一名学生（如：高一1班 - 张三）。
2.  **反馈**：下方的表格会立即刷新，显示该名学生的所有课程安排。

### 功能 B：添加/修改课程（两种方式）
*   **方式一（点击按钮）**：点击右上角 **“+ 添加/编辑课程”** 按钮。
*   **方式二（直观操作）**：直接点击表格中**任意一个格子**（例如周三第3节）。

**操作流程**：
1.  **弹出窗口**：显示“编辑课程安排”模态框。
    *   *注：如果是通过点击格子打开，星期和节次会自动填好。*
2.  **填写/修改**：
    *   **科目**：下拉选择科目（如：物理）。如果选择“(空闲)”，则表示删除该节课。
3.  **保存**：点击右下角的**蓝紫色 “保存”** 按钮。
4.  **反馈**：表格对应的格子立即更新显示新科目。

---

## 7. 学生考勤模块
### 界面外观
*   顶部操作栏（从左至右）：
    1.  **学生下拉框**：选择要查询的学生。
    2.  **月份选择框**：选择具体的年/月。
    3.  **蓝色 “查询” 按钮**。
    4.  最右侧：**绿色 “+ 新增考勤” 按钮**。
*   主体区域：考勤记录列表表格（包含日期、签到时间、签退时间、状态、操作）。

### 功能 A：查询考勤
1.  **选择条件**：在下拉框选择学生，在日期框选择月份。
2.  **点击查询**：点击**蓝色 “查询”** 按钮。
3.  **反馈**：
    *   如有数据：列表显示详细记录。
        *   *状态说明*：如果签到时间晚于 08:00，状态栏会自动显示**黄色“迟到”**标签，否则显示**绿色“正常”**。
    *   如无数据：列表下方显示文字“该月无考勤记录”。

### 功能 B：新增考勤
1.  **点击按钮**：点击右上角**绿色 “+ 新增考勤”** 按钮。
2.  **弹出窗口**：显示“录入考勤”模态框。
3.  **填写信息**：
    *   **学生**：默认选中当前查询的学生（也可更改）。
    *   **日期**：选择具体日期。
    *   **时间**：填写签到（如 07:55）和签退时间。
4.  **保存**：点击右下角**绿色 “保存”** 按钮。
5.  **反馈**：列表自动刷新，显示新添加的记录。

### 功能 C：编辑/删除考勤
1.  **点击编辑**：在表格每一行的最右侧，点击蓝色的 **“编辑”** 文字链接。
2.  **弹出窗口**：窗口内会加载该条记录的数据。
3.  **修改**：修改时间后点击**绿色 “保存”**。
4.  **删除**：
    *   在编辑窗口左下角会出现一个**浅红色背景的 “删除” 按钮**。
    *   点击后确认提示，即可删除该条考勤记录。
